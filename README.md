# primrootFinder
This program will have user input that performs various functions leading up to the final function which is to list all primitive roots for a given number

## List of Functions:
   0 - gcd
   
   1 - isPrime
   
   2 - getPrimeFactors
   
   3 - eulerPhi
   
   4 - orderMod
   
   5 - getPrimeFactors
   
   6 - primRootCalc
   
   7 - findPrimRoots
   
   8 - extraCredit
